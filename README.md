# 不要说挑战 🎴

一个简单有趣的线上抽牌网页，适合团队活动使用。

## 玩法说明

1. 点击"开始抽卡"按钮
2. 将手机屏幕面向他人，等待 3 秒
3. 对方会看到你的挑战内容（如"说绩效"、"手托脸"、"笑"）
4. 开始游戏：轮流说工作烦恼，过程中不能做卡片上的事！

## 词库内容

- **语言类（28张）**：说绩效、说排班、说用户... 说某个词就中招！
- **动作类（28张）**：手托脸、摸头、摇头... 做了这个动作就中招！
- **表情类（4张）**：笑、翻白眼... 做了这个表情就中招！

共 60 张卡片，纯随机抽取，每次都有新体验。

## 本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 推送代码到 main 分支
3. 在仓库 Settings → Pages 中启用
4. 访问 `https://<username>.github.io/<repo-name>/`

## 技术栈

- 纯前端：单个 HTML 文件
- 无需构建工具
- 响应式设计，适配手机竖屏
